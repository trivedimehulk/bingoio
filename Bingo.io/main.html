How to Implement Speech Fill

Step 1:

Add bingo button at the bottom of your HTML. E.g.


<button id="bingoButton">Button</button>
Step 2:

Add bingo style in HEAD section of your HTML. E.g.


<link rel="stylesheet" href="bingoButton.css">
Step 3:

Create and inject fieldMap.js like below in your HTML file at the end preferred.

[Make sure to read our instruction manual to understand how to create an easy field map.]


<script src="fieldMap.js"></script>
Step 4:

Include the Bingo magic JavaScript file at the end preferred.


<script src="magic.js"></script>
